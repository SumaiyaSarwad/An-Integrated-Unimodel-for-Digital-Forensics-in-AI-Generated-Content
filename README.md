# An-Integrated-Unimodel-for-Digital-Forensics-in-AI-Generated-Content

view this link to explore the frontend of this project :- https://sumaiyasarwad.github.io/unimodel-trial/

step 1:- download python version 3.10.11                                                                                                                                  
step 2:- download zip file                                                                                                                                                                   
step 3:- exctract file                                                                                                                                                                                                 
step 4:- import to VScode                                                                                                                                                                
step 5:- open new terminal inside VScode                                                                                                                                                                
step 6:- redirect to the project path as given below                                                                                                                                                                            
cd ai_detector_unimodel  
python -m venv venv  
venv\Scripts\activate
<img width="629" height="201" alt="image" src="https://github.com/user-attachments/assets/8d29ded7-c80a-4ced-8e64-6f0445da257d" />

step 7:- after creating and activating environment it will look like in above image (venv)                                                                                                                                               
now install the requirement using the below commands one by one                                                                                                                            
pip install pandas scikit-learn nltk joblib numpy                                                                                                                                                
pip install language-tool-python                                                                                                                                                            
pip install pyspellchecker nltk                                                                                                                                                  
pip install -r requirements.txt                                                                                                                                                                            
step 8:- once after completing the installation of requrements run this given below command to run project                                                                                    
python app.py

after the project installtion process use given below commands to run project everytime                                                                                                      
cd ai_detector_unimodel                                                                                                                                                              
venv\Scripts\activate                                                                                                                                                                                                          
python app.py
